📌 Requirements

  • File format must be .xlsx or .xls
  • File size must not exceed 5MB
  • Maximum 500 rows per import
  • All required fields must be filled
  • Staff ID and Email must be unique (not already in system)
  • Dates must be in YYYY-MM-DD or DD/MM/YYYY format
  • Department, Designation, and other relationships must exist in the system
